Fused Brain Image and Its Generation Using MATLAB

A fused brain image combines both structural and functional brain images to provide a comprehensive view of the brain’s anatomy and activity. This technique is useful in medical diagnostics, helping doctors to analyze both the anatomical and functional aspects of the brain.

Generating a Fused Brain Image Using MATLAB

To create a fused brain image in MATLAB, we followed these steps:

  • 1. Loading Structural and Functional Brain Images
    We loaded both the structural MRI scan and the functional brain image to overlay the activity regions on the structural image.
  • 2. Aligning and Normalizing the Images
    Both images were resized and normalized to ensure proper alignment, with the functional activity map adjusted to match the intensity levels of the structural image.
  • 3. Applying Fusion Techniques
    A weighted averaging method was applied to blend the structural and functional layers, allowing both anatomical details and activity regions to be visible.
  • 4. Displaying the Final Fused Image
    The final fused brain image was displayed using MATLAB’s imshow() function, showing a combination of structural and functional features.

MATLAB Code to Generate the Fused Brain Image

% Load the structural brain image (previously created)
load mri; % Built-in MRI data
D = squeeze(D); % Convert the 4D data to 3D
structuralImage = mat2gray(D(:,:,15)); % Slice 15 for structural image

% Create the functional brain image (same as before)
functionalImage = mat2gray(D(:,:,15)); % Slice 15

% Simulate functional activity regions
[xGrid, yGrid] = meshgrid(1:size(functionalImage, 2), 1:size(functionalImage, 1));
activity1 = ((xGrid - 40).^2 + (yGrid - 60).^2) <= 30; % First activity region
activity2 = ((xGrid - 80).^2 + (yGrid - 100).^2) <= 20; % Second activity region
activity3 = ((xGrid - 50).^2 + (yGrid - 140).^2) <= 25; % Third activity region

% Add functional activity to the functional image
functionalImage(activity1) = 0.8; % Activity level 1
functionalImage(activity2) = 0.9; % Activity level 2
functionalImage(activity3) = 0.85; % Activity level 3

% Enhance the structural image by detecting edges (using Sobel filter)
structuralEdges = edge(structuralImage, 'Sobel'); % Highlight edges
structuralEnhanced = mat2gray(imoverlay(structuralImage, structuralEdges, [1 1 1])); % Combine edges with structural image

% Fuse the images by overlaying functional activity onto structural image
fusedImage = structuralEnhanced; % Start with the structural image
fusedImage(activity1) = 0.7; % Add functional activity at region 1
fusedImage(activity2) = 0.9; % Add functional activity at region 2
fusedImage(activity3) = 0.8; % Add functional activity at region 3

% Display the fused image
imshow(fusedImage);
